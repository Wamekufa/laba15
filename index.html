Лабораторная работа №15: Классы и наследование в JavaScript

Демонстрация классов и наследования

Нажмите на кнопку для создания объекта и вывода информации

Иерархия классов:

Person (базовый класс) → Student, Teacher, Employee

Каждый класс расширяет базовый класс Person, добавляя свои свойства и методы.

Примеры кода

Класс Person:

class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }
    
    introduce() {
        return `Меня зовут ${this.name}, мне ${this.age} лет.`;
    }
}

Класс Student (наследует Person):

class Student extends Person {
    constructor(name, age, major) {
        super(name, age);
        this.major = major;
    }
    
    introduce() {
        return `${super.introduce()} Я учусь на специальности ${this.major}.`;
    }
}
